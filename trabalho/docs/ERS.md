
# Software Requirements Specification (ERS)

## Requisitos Funcionais

1. **Cadastro de Kitnets**: O sistema deve permitir que os administradores cadastrem novas kitnets com nome, endereço e preço.
2. **Autenticação de Usuários**: O sistema deve autenticar os usuários utilizando JWT.
3. **CRUD de Kitnets**: O sistema deve permitir que os administradores criem, leiam, atualizem e excluam kitnets.

## Requisitos Não Funcionais

1. **Segurança**: O sistema deve garantir a segurança das informações com criptografia de senhas.
2. **Desempenho**: O sistema deve ser capaz de responder rapidamente a requisições de CRUD de Kitnets.
3. **Escalabilidade**: O sistema deve ser modular e escalável para acomodar mais funcionalidades no futuro.
