
# Business Requirements Document (BRD)

## Requisitos de Negócio

1. **Cadastro de Kitnets**: O sistema deve permitir o cadastro de kitnets com informações como nome, endereço e preço.
2. **Autenticação de Usuários**: O sistema deve permitir que os usuários se cadastrem e façam login utilizando autenticação JWT.
3. **CRUD de Kitnets**: O sistema deve permitir a criação, leitura, atualização e exclusão de kitnets.
4. **Segurança**: O sistema deve proteger as informações dos usuários com criptografia e autenticação segura.
5. **Documentação de API**: A API deve ser bem documentada com o Swagger para que outros desenvolvedores possam testar os endpoints facilmente.
