# Planejamento de Atividades

## Fases

1. Levantamento de requisitos (BRD, SRS)
2. Modelagem (diagramas)
3. Configuração do ambiente (.NET, Docker, EF)
4. Implementação da API
5. Testes automatizados
6. Documentação e Evidências

## Cronograma Exemplo

| Fase | Atividade | Entregável |
|------|-----------|------------|
| 1 | Requisitos | BRD, SRS |
| 2 | Modelagem | Diagramas UML/ER |
| 3 | Ambiente | Docker + projeto .NET |
| 4 | Implementação | API funcionando com Swagger |
| 5 | Testes | Projeto xUnit |
| 6 | Evidências | Documento com prints e descrição |
